# Data Layer

This folder is reserved for public, reproducible and non-confidential datasets.

## Data policy

- No proprietary banking data
- No confidential client data
- No private model outputs
- No synthetic data presented as real market data
- Public or clearly illustrative data only

## Current use

The current repository version focuses on model-validation logic and uses small internal examples for testing. Future versions may add public market datasets for VaR, stress testing, volatility monitoring and high-dimensional financial data checks.
