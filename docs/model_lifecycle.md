# Model Lifecycle

A simplified model lifecycle contains the following stages:

1. Model proposal
2. Methodology design
3. Implementation
4. Data validation
5. Independent review
6. Approval or rejection
7. Monitoring
8. Periodic review
9. Issue remediation
10. Retirement or replacement

## Lifecycle logic

A model can be technically correct and still become weak if data changes, market structure changes, assumptions break down or implementation logic drifts from the documented methodology.
