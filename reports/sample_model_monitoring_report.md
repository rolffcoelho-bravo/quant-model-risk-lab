# Sample Model Monitoring Report

## Model

Historical VaR Model

## Monitoring objective

Track whether model outputs remain plausible under recent market behaviour.

## Monitoring checks

- Number of observations
- Missing values
- VaR threshold
- Number of exceptions
- Exception rate
- Stress-period behaviour

## Example interpretation

A rising exception rate may indicate that the model is no longer capturing current market risk adequately. This would require review of the lookback window, volatility regime, data quality and stress assumptions.
